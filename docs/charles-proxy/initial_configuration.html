Начальная конфигурация

Установка​

Обратите внимание

В настоящем руководстве будет рассмотрена установка, настройка (и, соответственно, эксплуатация) Charles Proxy версии 5.0.1 для Windows (последней, по состоянию на дату составления настоящего руководства).

За вычетом ряда несущественных платформенных отличий, данное руководство может быть применено в том числе и к версиям для macOS или Linux.

Актуальная версия Charles Proxy для Windows поставляется в формате APPX. Установка APPX на Windows 11 практически не отличается от установки приложений, упакованных в .msi или .exe: достаточно дважды кликнуть по установочному файлу и вы увидите окно установщика.

Первый шаг установки

Когда приложение будет установлено необходимо запустить его (если оно не запустилось само, вследствие неснятия галочки в установщике) и разрешить работу приложения через Брандмауэр Windows.

Запрос на разрешение работы через Windows Firewall

После ответа на запрос приложения запустится и откроет главное окно.

Главное окно приложения

Начальная конфигурация​

Вы могли обратить внимание на то, что левая часть главного окна приложения уже содержит определённые запросы.

Это связано с тем, что по-умолчанию приложение запускается в режиме "системного прокси" и весь HTTP-трафик вашего приложения начинает отслеживаться приложением.

Такое поведение может быть желательным, однако если вы планируете использовать Charles Proxy главным образом для тестирования мобильных приложений, то вы можете захотеть отключить запуск "системного прокси" при старте приложения.

  • Отключение для текущей сессии
  • Отключение запуска по умолчанию

Чтобы отключить данную настройку в рамках текущей сессии, необходимо открыть меню Proxy и снять соответствующую галочку с пункта Windows Proxy.

Данная операция повлияет только на текущую сессию работы с приложением и после его перезапуска "системное прокси" вновь станет активным.

Данная операция позволит отключить запуск "системного прокси" при старте приложения.

Настройка осуществляется в два этапа.

  • Шаг 1
  • Шаг 2

Откройте настройки прокси в меню Proxy → Proxy settings...

Перейдите на вкладку Windows, снимите галочку около пункта Enable Windows proxy on launch и сохраните настройки нажатием на кнопку Done

Если вы отключили запуск системного прокси при старте приложения, то вы всегда можете включить его для отдельной сессии посредством установки галочки у пункта Windows Proxy в меню Proxy.

Регистрация лицензии​

Было бы упущением не напомнить о необходимости зарегистрировать вашу копию Charles Proxy.

Регистрация приложения производится в два этапа.

  • Шаг 1
  • Шаг 2
  • Проверка лицензии

Для ввода данных лицензии откройте соответствующее окно, обратившись по пути Help → Enter Charles license...

Обратите внимание

После подтверждения посредством нажатия кнопки Register приложение поблагодарит за регистрацию лицензии и закроется.
Повторный запуск приложения необходимо произвести самостоятельно.

Введите имя владельца лицензии и лицензионный ключ.

По желанию, после запуска приложения, вы можете проверить наличие лицензии открыв меню Help.
Вместо Enter Charles license... в первом пункте меню будет отображено имя владельца лицензии.
